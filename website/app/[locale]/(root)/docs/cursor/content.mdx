## Basic usage
### Set the cursor style
Use the `cursor:pointer` to change the mouse cursor when hovering an element.

(hover) Hovering the block to see the cursor style

## Conditionally apply
Apply styles based on different states using [selectors](/docs/state-selectors) and [conditional queries](/docs/conditional-queries).
```html
<div class="cursor:pointer**:hover** cursor:pointer**@sm** cursor:pointer**@dark** cursor:pointer**@print**">â€¦</div>
```
