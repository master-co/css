## Basic usage
### Block & Inline
Use `block`, `inline`, and `inline-block` to set the flow of elements.
<Demo $py={0}>
    <div className='bg:white@light bg:gray-20@dark p:30'>
        <p>
            Lorem ipsum dolor sit amet, consectetur adipiscing elit.
            <span className="box block! bg:slate-95!@light">
                `display:block`
            </span>
            Nulla et elit dictum, tempor augue quis, rhoncus enim. Nunc lacinia, velit vel convallis tincidunt, ante nisi maximus nunc, at aliquam nisi lectus in mauris. Nam eros sem, ullamcorper nec mollis nec, imperdiet a metus. Nunc pulvinar tortor ipsum,
            <span className="box inline! my:8 bg:slate-95!@light">
                `display:inline`
            </span>
            nec malesuada tellus posuere id. Nullam sed nulla eget ante ultrices tristique. Orci varius natoque penatibus et magnis dis parturient montes,
            <span className="box inline-block! my:8 bg:slate-95!@light">
                `display:inline-block`
            </span>
            nascetur ridiculus mus. Proin sodales turpis ut dolor posuere, nec aliquet nulla dapibus.
        </p>
    </div>
</Demo>
```html
<div>

    </span>
```

</Demo>

### Contents
Use `contents` to make the element&rsquo;s children like direct children of the element&rsquo;s parent, ignoring the element itself.
<Demo>
    <div className="flex gap:20 w:full">
        <div className="flex:1 box">01</div>
        <div className="contents">
            <div className="box flex:1">02</div>
            <div className="box flex:1">03</div>
        </div>
        <div className="flex:1 box">04</div>
    </div>
</Demo>
```html
<div class="flex gap:20">
    <div class="flex:1">01</div>
    <div class="contents">
        <div class="flex:1">02</div>
        <div class="flex:1">03</div>
    </div>
    <div class="flex:1">04</div>
</div>
```

### Flow Root
Use `flow-root` to create a block-level element that establishes a new block formatting context.
{<Demo $py={0}>
    <div className="bg:white@light bg:gray-20@dark p:30 w:full">
        <div className="w:full flow-root bg-stripe-pink">
            <Image src="/images/blur.png" className="r:8 object:cover my:0 mr:30 float:left" width={160} height={90} alt="Float Image" />
            <p className="my:0">Lorem ipsum dolor sit amet, consectetur adipiscing elit.</p>
        </div>
        <p>Use `flow-root` to clear floats.</p>
    </div>
</Demo>}
```html
<div class="flow-root">
    <img class="float:left" …/>
    <p>Lorem ipsum dolor sit amet… </p>
</div>
Use flow-root to clear floats
```

Use `list-item` to create a block that behave like a list item.

### Hidden
Use `hide` to hide the element. The element is not rendered in the document.
<Demo>
    <div className="flex gap:20 w:full flex:1_:where(div)">
        <div className="box">01</div>
        <div className="box hide!">02</div>
        <div className="box">03</div>
    </div>
</Demo>
```html
<div class="flex gap:20">
    <div>01</div>
    <div class="hide">02</div>
    <div>03</div>
</div>
```

## Conditionally apply
Apply styles in [different states](/docs/states-and-queries) using selectors, media queries, etc.
```html
<div class="display:block**:hover** display:block**@sm** display:block**@dark** display:block**@print**">…</div>
```
