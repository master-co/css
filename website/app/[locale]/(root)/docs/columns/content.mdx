## Basic usage

### Flexible multi-column layout
Use `mcss cols:count|width` to set the minimum width of each column to `width` and the maximum number of columns to `count` columns when creating a flexible multi-column layout.

## Conditionally apply
Apply styles based on different states using [selectors](/docs/state-selectors) and [conditional queries](/docs/conditional-queries).
```html
<div class="columns:5**:hover** columns:5**@sm** columns:5**@dark** columns:5**@print**">…</div>
```
