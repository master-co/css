## Quick start
This is a simple Vite example of Vue.js server-side rendering.

A full-stack framework, such as [Nuxt.js](/docs/installation/nuxtjs), is recommended for better application performance and experience.
<StepSection>
    <Step $row>
        <StepL>
            ### <StepNum />Clone the example
            Copy-paste the following commands to quickly start using the [vuejs.pr.beta.css.master.co](https://vuejs.pr.beta.css.master.co).

            You can skip all [installation](#installation) steps.
        </StepL>
        <StepR>
            ```bash name=Terminal
            npx degit master-co/css/examples/vue.js-with-progressive-rendering __my-project__
            cd __my-project__
            npm install @master/css.vue@beta
            npm install
            npm run dev
            ```
        </StepR>
        <StepEnd/>
    </Step>
</StepSection>
