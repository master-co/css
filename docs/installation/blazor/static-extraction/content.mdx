## Quick start
<StepSection>
    <Step $row>
        <StepL>
            ### <StepNum />Clone the example
            Copy-paste the following commands to quickly start using the [blazor.se.beta.css.master.co](https://blazor.se.beta.css.master.co).

            You can skip all [installation](#installation) steps.
        </StepL>
        <StepR>
            ```bash name=Terminal
            npx degit master-co/css/examples/blazor-with-static-extraction __my-project__
            cd __my-project__
            npm install @master/css-extractor.blazor@beta
            npm install
            npm run dev
            ```
        </StepR>
        <StepEnd/>
    </Step>
</StepSection>

## Installation

<StepSection>
    <Step $row>
        <StepL>
            ### <StepNum />Create a project
            If you don't have a Blazor project, create one first. It's recommended to refer to [Getting Started - Blazor](https://blazorjs.dev/guide/)
        </StepL>
        <StepR>
            <CodeTabs>
                {[
                    {
                        name: 'Terminal',
                        lang: 'bash',
                        code: `
                            npm create blazor@latest project -- --template vanilla-ts
                            cd project
                            npm install
                        `
                    }
                ]}
            </CodeTabs>
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Install Master CSS Extractor
            Install the Master CSS extractor into your project via package managers.
        </StepL>
        <StepR>
            ```install
            **@master/css-extractor.blazor@beta**
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Initialize configuration file
            Run `npx mcss init` to create a configuration file [master.css.ts](/docs/configuration).
        </StepL>
        <StepR>
            ```bash name=Terminal
            npx mcss init
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Set up CSS extractor
            - Create a `blazor.config.ts` file
            - Add a `CSSExtractorPlugin` to the blazor `plugins`
        </StepL>
        <StepR>
            ```js name=blazor.config.ts
            +import { defineConfig } from 'blazor'
            +import { CSSExtractorPlugin } from '@master/css-extractor.blazor'

            +export default defineConfig({
            +    plugins: [
            +        CSSExtractorPlugin()
            +    ]
            +})
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Import virtual CSS module
            Import the virtual CSS module `.virtual:master.css` into the `main.ts`.
        </StepL>
        <StepR>
            ```ts name=main.ts
            …
            +import '.virtual:master.css';
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Launch server
            Run the Blazor development server.
        </StepL>
        <StepR>
            ```bash name=Terminal
            npm run dev -- --open
            ```
        </StepR>
    </Step>
    <Step>
        ### <StepNum />Hello world!
        Now style your first element using Master CSS syntax!
        ```html name=index.html
        <!DOCTYPE html>
        <html lang="en" style="display: none">
            …
            <body>
        +        <h1 class="font:40 font:heavy italic m:50 text:center">Hello World</h1>
                …
            </body>
        </html>
        ```
        <p className="mt:50">Open your browser to watch the changes.</p>
        <HelloWorld url="localhost:5173" />
    </Step>
</StepSection>
