## Quick start
<StepSection>
    <Step $row>
        <StepL>
            ### <StepNum />Clone the example
            Copy-paste the following commands to quickly start using the [svelte.se.beta.css.master.co](https://svelte.se.beta.css.master.co).

            You can skip all [installation](#installation) steps.
        </StepL>
        <StepR>
            ```bash name=Terminal
            npx degit master-co/css/examples/svelte-with-static-extraction __my-project__
            cd __my-project__
            npm install @master/css-extractor.vite@beta
            npm install
            npm run dev -- --open
            ```
        </StepR>
        <StepEnd/>
    </Step>
</StepSection>

## Installation

<StepSection>
    <Step $row>
        <StepL>
            ### <StepNum />Create a project
            If you don't have a Svelte project, create one first. It's recommended to refer to [Svelte Kit](https://kit.svelte.dev/).
        </StepL>
        <StepR>
            ```bash name=Terminal
            npm create svelte@latest __project__
            cd __project__
            npm install
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Install Master CSS
            Install the Master CSS extractor into your project via package managers.
        </StepL>
        <StepR>
            ```install
            **@master/css-extractor.vite@beta**
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Initialize configuration file
            Run `npx mcss init` to create a configuration file [master.css.ts](/docs/configuration).
        </StepL>
        <StepR>
            ```bash name=Terminal
            npx mcss init
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Set up CSS extractor
            - Create a `vite.config.ts` file
            - Add a `CSSExtractorPlugin` to the vite `plugins`
        </StepL>
        <StepR>
            ```ts name=vite.config.ts
            +import { defineConfig } from 'vite'
            +import { CSSExtractorPlugin } from '@master/css-extractor.vite'

            +export default defineConfig({
            +    plugins: [
            +        CSSExtractorPlugin()
            +    ]
            +})
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Import virtual CSS module
            Import the virtual CSS module `.virtual/master.css` into the `src/routes/+layout.svelte`.
        </StepL>
        <StepR>
            ```svelte name=src/routes/%2Blayout.svelte
            <script>
                import Header from './Header.svelte';
                import './styles.css';
            +    import '.virtual/master.css';
            </script>
            â€¦
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Launch server
            Run `npm run dev -- --open` to start your Vite development server
        </StepL>
        <StepR>
            ```bash name=Terminal
            npm run dev -- --open
            ```
        </StepR>
    </Step>
    <Step>
        ### <StepNum />Start using Master CSS
        Now style your first element using Master CSS syntax!
        ```svelte name=src/routes/%2Bpage.svelte
        +<h1 class="font:40 fg:blue font:heavy italic m:50 text:center">Hello World</h1>
        ```
        <p className="mt:50">Open your browser to watch the changes.</p>
        <HelloWorld url="localhost:5173" />
    </Step>
</StepSection>
